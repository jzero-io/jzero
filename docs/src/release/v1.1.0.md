## 新增特性

### cmd/jzero

* jzero gen 时对 desc 自动生成 metadata.json 文件放入 ~/.jzero/desc-metadata 文件夹中, 主要是为支持 goland 插件能从 logic 跳转到 api/proto 文件中

```json
{
  "api": {
    "routes": [
      {
        "handler": "Create",
        "group": "user",
        "path": "post:/",
        "logic": "/Volumes/jaronnie_volume/gocloudcoder/github/jzero-io/examples/simpleapi/internal/logic/user/create.go",
        "desc": "/Volumes/jaronnie_volume/gocloudcoder/github/jzero-io/examples/simpleapi/desc/api/user.api",
        "desc-line": 16
      },
      {
        "handler": "Version",
        "group": "version",
        "path": "get:/api/version",
        "logic": "/Volumes/jaronnie_volume/gocloudcoder/github/jzero-io/examples/simpleapi/internal/logic/version/version.go",
        "desc": "/Volumes/jaronnie_volume/gocloudcoder/github/jzero-io/examples/simpleapi/desc/api/version.api",
        "desc-line": 21
      }
    ]
}
```

## 更新步骤

jzero upgrade --channel v1.1.0
