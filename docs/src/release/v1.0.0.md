From the creation of jzero project on 2024.04.07 to 2026.01.01, after more than 600 days of refinement, jzero has finally released v1.0.0 stable version.

Includes the following important components:

* jzero command-line tool (jzero)
* jzero core library (core)
* jzero-admin (backend management system based on jzero, as jzero's best practice implementation)
* jzero-intellij (goland plugin adapted for jzero, greatly enhancing development experience)
