## New Features

### cmd/jzero

* Added support for setting model cache expiry, allowing fine-grained control over cache TTL for improved performance management [#397](https://github.com/jzero-io/jzero/pull/397)

* Refactored genrpc and genzrpcclient for improved code generation quality and maintainability [#389](https://github.com/jzero-io/jzero/pull/389)

## Bug Fixes

### cmd/jzero

* Fixed anomalies when generating swagger and mono modes to ensure consistent code generation [#404](https://github.com/jzero-io/jzero/pull/404)

* Fixed issue with add api command [#426](https://github.com/jzero-io/jzero/pull/426)

## Documentation

* Enhanced documentation for proto, serverless, and template guides with more detailed examples and explanations [#398](https://github.com/jzero-io/jzero/pull/398)

* Added English documentation support for international users

* Updated README and skills documentation
