## 新增 api 文件

将在 desc/api 文件夹下新增 api 文件

```shell
# group 为 test
jzero add api test
# group 为 test/test1
jzero add api test/test1
```

## 新增 proto 文件

将在 desc/proto 文件夹下新增 proto 文件

```shell
# Service 为 Test
jzero add proto test
# Service 为 TestTest1
jzero add proto test/test1
```

## 新增 sql 文件

将在 desc/sql 文件夹下新增 sql 文件

```shell
# table 名为 test
jzero add sql test
```
