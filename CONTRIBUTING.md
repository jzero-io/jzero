# Contributing to Jzero

## steps

### 1. fork

https://github.com/jzero-io/jzero/fork

### 2. clone

```shell
git clone https://github.com/your_username/jzero
```

### 3. checkout branch

```shell
cd jzero

git checkout -b feat/patch-1
```

### 4. push

```shell
git add .
git commit -m "feat(xx): custom message"
git push
```
